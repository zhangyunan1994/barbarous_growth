		元空间(Meta Space)
			* 代替了原来的永久区(PremGen)
			* 直接使用的是物理内存而不是JVM分配的内存
			* 说白了,你物理内存多大,这个空间就能有多大
			* 设置参数
				-XX:MetaspaceSize 
				-XX:MaxMetaspaceSize
			* 但如果不指定MaxMetaspaceSize的话，Metaspace的大小仅受限于native memory(物理内存)的剩余大小。
			* 也就是说永久代的最大空间一定得有个指定值，而如果MaxPermSize指定不当，就会OOM(内存耗尽)。
